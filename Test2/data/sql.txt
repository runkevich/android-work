Совмещает две таблицы в один результат
______________________________________________________________
Ответ на вопрос задаваемый на собесе, как смэрджить таблицы?
 двойное равно = одинаркное !
INNER JOIN возвращает только те даныне,которые совместились !!!

SELECT * FROM user INNER JOIN country ON user.id_c = country.id

________________________________________________________________________________________
Совмещает две таблицы в один результат, то что не совмещается будет взята левая таблица,
а в правой будут ноли. (Слева, что совместилось, то осталось, справа  - уничтожается)

SELECT * FROM user LEFT JOIN country ON user.id_c = country.id


_________________________________________________________________________________________
Совмещает две таблицы в один результат, то что не совмещается будет взята правая таблица,
а в левой будут ноли.(Справа, что совместилось, то осталось, слева  - уничтожается)
